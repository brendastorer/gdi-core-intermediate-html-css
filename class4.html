Media Queries

  • Conditional style sheets
  • Changing specific CSS styles on different browsers based on width, device-pixel-ratio, and orientation
  • Called via a meta tag

Media Queries

A media type and at least one expression that limits the style sheets' scope by using media features, such as width, height, and color

Media Queries

<meta name="viewport" content="width=device-width, initial-scale=1">
@media (min-width: 801px) {
  html {
    background: darkcyan;
  }
}
@media (min-width: 401px) and (max-width: 800px) {
  html {
    background: khaki;
  }
}
@media (max-width: 400px) {
  html {
    background: salmon;
  }
}

Let's Develop It

  • All of the content should fit within the viewport on all screen sizes.
